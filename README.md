# Botflix

## Descrição do Projeto
Este projeto foi desenvolvido como parte de uma aula para relembrar as práticas de HTML, CSS e JavaScript. O objetivo principal foi criar uma interface simples e funcional que demonstra o uso dessas tecnologias.

## Tecnologias Utilizadas
- **HTML**: Estrutura básica do site.
- **CSS**: Estilização e layout, utilizando arquivos como `animations.css`, `reset.css`, `responsivo.css` e `styles.css`.
- **JavaScript**: Interatividade e funcionalidades, implementadas no arquivo `index.js`.

## Aprendizados
Durante o desenvolvimento deste projeto, aprendi sobre o n8n, uma ferramenta de automação que permite integrar diferentes serviços e criar fluxos de trabalho automatizados.
